# Vimide
Set up vim plugin to make vim like IDE by one install

It was just made for me!

1. Plugin List
    a. Vundle
    b. The NERD Tree
    c. Tagbar
    d. Vim-airline
    e. Vim-airline-themes
    f. vim-colors-solarized
    g. syntastic
    h. vimproc
    i. vimshell
    j. youcompleteme
    k. vim-indent-guides
    l. vim-diminactive
  
2. Install
Run install.sh
./install.sh
